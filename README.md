# DarkKnight First Unity Project

## Project Overview
This repository contains a 2D action game prototype developed with **Unity** and **C#**.

The project focuses on building a small playable combat experience with core action game mechanics, including player movement, attacks, enemy behavior, basic combat feedback, UI, and boss-related features.

This project was created as part of my game development / programming practice and demonstrates my work with Unity, C#, animation systems, combat logic, and gameplay integration.

---

## Main Features
- 2D player movement
- Jump system
- Attack system
- Enemy detection and chasing behavior
- Melee / ranged enemy combat behavior
- Boss combat prototype
- Dash / dodge mechanic
- Health UI and damage system
- Death and respawn logic
- Combat feedback effects such as hit reaction, screen shake, and sound integration
- Animation control using Unity Animator

## Project Structure
Typical important folders in this repository include:
- `Assets/` – Unity assets, scripts, animations, audio, prefabs, scenes
- `ProjectSettings/` – Unity project settings
- `Packages/` – Unity package configuration

Game scripts are mainly written in **C#** and located under the Unity project asset folders.

## How to Run
1. Open the project using **Unity Hub**
2. Use the correct Unity version for this project
3. Open the main scene in the `Assets` folder
4. Press **Play** in Unity Editor to run the game
